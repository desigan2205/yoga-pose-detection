🧘 Yoga Pose Detection and Correction
📌 Overview

This project uses Deep Learning and Computer Vision to detect yoga poses and provide pose correction feedback.

✨ Features
Detects yoga poses in real time
Provides posture correction
Uses deep learning models
User-friendly interface

🛠 Technologies
Python
TensorFlow
OpenCV
MediaPipe

🚀 How to Run
pip install -r requirements.txt
python app.py

📂 Project Structure
app.py – Main application
model_train.py – Model training
model_predict.py – Prediction
pose_correction.py – Pose correction
